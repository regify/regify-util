◆ ruSetNew()

RUAPI ruSet ruSetNew ( ruHashFunc  hash,
ruMatchFunc  match,
ruFreeFunc  itemFree,
uint32_t  expectedSize 
)

Creates a new thread safe set object. This is the long but flexible version to create a hash set.

Parameters
hashA required function that returns a hash for the given key to be used internally to determine bucket assignments.
matchA required function that determines whether the 2 given keys are equal.
itemFreeAn optional function to be called when a key is to be finalized (deleted).
expectedSizeAn optional or 0 initial estimate of how many members the set will have.
Returns
The newly created set object to be freed with ruSetFree or NULL if required arguments weren't given.
