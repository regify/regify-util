◆ ruTypeInt64()

RUAPI ruType ruTypeInt64 ( void  )

Returns an ruType used for 64 bit integers.

Example:

// error checking left out for brevity
int64_t k1 = 23, v1 = 42, r = 0;
ruMap rm = ruMapNew(ruTypeInt64(), ruTypeInt64());
ruMapPut(rm, &k1, &v1);
ruMapGet(rm, &k1, &r);
printf("k: %ld v: %ld\n", k1, r);
ruMapFree(rm);
