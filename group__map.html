RUAPI ruMap ruMapNew ( ruHashFunc  hash,

ruFreeFunc  valFree,
uint32_t  expectedSize 
)

Creates a new thread safe map object. This is the long but flexible version to create a hash map.

Parameters
hashA required function that returns a hash for the given key to be used internally to determine bucket assignments.
matchA required function that determines whether the 2 given keys are equal.
keyFreeAn optional function to be called when a key is to be finalized (deleted).
valFreeAn optional function to be called when a value is to be finalized (deleted).
expectedSizeAn optional or 0 initial estimate of how many members the map will have.
Returns
The newly created map object to be freed with ruMapFree or NULL if required arguments weren't given.
